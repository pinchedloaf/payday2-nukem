payday2-nukem
---------------
Payday 2 mod to nuke all enemies on map.

Requires
--------
BLT 1.07 (r11_r5)
- https://github.com/JamesWilko/Payday-2-BLT/releases/tag/1.07

Installation
------------
- Copy to mods directory after setting up BLT.
- Setup hotkeys for BLT.

Notes
-----
Payday 2, Update 93+ (2015-12-18)
